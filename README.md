# Windows 游戏优化助手

一键优化 Windows 系统设置，提升游戏性能。

## 功能

- 关闭非必要后台进程
- 调整电源计划为高性能的电源计划
- 清理文件

## 下载与使用

1. 前往 [Releases] 下载最新版
2. 右键以管理员身份运行
3. 点击“一键优化”

## 系统要求

Windows 10 / 11（64位）

## 注意

本工具目前仅支持中文界面。
